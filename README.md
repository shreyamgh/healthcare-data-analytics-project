# healthcare-data-analytics-project
End-to-end healthcare data analysis using Python, SQL, and Tableau
Overview
This project focuses on analyzing healthcare data to uncover insights related to patient demographics, medical conditions, hospital trends, and billing patterns. The goal is to simulate real-world healthcare analytics using industry-standard tools.

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib)
SQL (for querying and data extraction)
Tableau (for interactive dashboard

📊 Dataset
Records: 55,500 patients
Features: Age, Gender, Medical Condition, Billing Amount, Admission Type, Hospital, etc.

🔍 Key Analysis Performed
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Trend analysis (monthly admissions)
Top medical conditions identification
Billing amount distribution analysis
